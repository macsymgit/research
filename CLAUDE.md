# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a research documentation repository containing comparative analysis of University of Waterloo architecture-related programs. All content is in Markdown format — there is no executable code, build system, or package manager.

## Content Overview

Three primary research documents (March 2026):

- **waterloo-architecture.md** — UWaterloo BAS + MArch program: co-op details, OAA licensing pathway (~9–11 years), career outcomes (~$85,300/yr median salary), Rome semester, AI impact analysis
- **waterloo-architectural-engineering.md** — UWaterloo BASc Architectural Engineering (5-year CEAB-accredited): 5 co-op terms, direct P.Eng. pathway (~3 years post-grad), entry salaries $62–75K, building systems specializations
- **waterloo-arch-comparison.md** — Side-by-side comparison: admission requirements, curriculum differences (design-led vs. engineering-led), salary trajectories, ROI analysis, 12-factor decision matrix

## Research Conventions

Documents follow a consistent structure: Table of Contents → Executive Summary → Program Details → Financial Analysis → Career Outcomes → Decision Frameworks → Source Citations.

All factual claims are cited to authoritative sources (UWaterloo, OAA, PEO, Statistics Canada, Job Bank, BuildForce Canada). When adding or updating content, maintain citation standards.

## Claude Code Settings

`.claude/settings.local.json` restricts WebFetch to specific authoritative domains (uwaterloo.ca, jobbank.gc.ca, peo.on.ca, buildforce.ca, etc.) to keep research sourced from credible references.
