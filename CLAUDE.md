# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a research documentation repository centered on the **McMaster University Honours Life Sciences (BSc) → medical school** pathway. All content is in Markdown format — there is no executable code, build system, or package manager.

## Content Overview

Primary research and planning documents (2026), each with a paired Chinese (`-zh.md`) translation:

- **mcmaster-life-sciences.md** — McMaster Honours Life Sciences (BSc) deep report: med school pipeline, MCAT/CASPer/MMI tactics, realistic admission probabilities, alternative health careers, year-by-year action plan
- **mcmaster-life-sci-to-med-school-plan.md** — Step-by-step G12-to-matriculation playbook for all 7 Ontario MD schools, with per-school 2026-27 admission rules, OMSAS roadmap, Plan B activation, and ROI analysis
- **mcmaster-life-sci-year1-schedule.md** / **mcmaster-life-sci-year2-schedule.md** — Term-by-term Year 1 & Year 2 course plans reconciled with the official Gateway + Honours Life Sci Level II requirements
- **mcmaster-life-sci-year1-action-items.md** — Checklist of admission, registration, and first-year tasks

## Research Conventions

Documents follow a consistent structure: Table of Contents → Executive Summary → Program Details → Financial Analysis → Career Outcomes → Decision Frameworks → Source Citations.

All factual claims are cited to authoritative sources (McMaster, OMSAS, AFMC, the 7 Ontario MD schools, Statistics Canada, Job Bank). When adding or updating content, maintain citation standards. Every English document has a paired Chinese (`-zh.md`) translation that must be kept in sync.

## Claude Code Settings

`.claude/settings.local.json` restricts WebFetch to specific authoritative domains to keep research sourced from credible references.
