## 1. 30秒摘要

| | **建筑学（BAS+MArch）** | **建筑工程学（BASc）** |
|--|-----|------|
| **核心定位** | 以设计为导向；您决定*建筑的外观与感受* | 以工程为导向；您确保*建筑的稳固、运转与性能* |
| **学位类型** | 预职业学士 BAS + 职业硕士 MArch | 工程学学士（BASc） |
| **总学制** | 6–7年（BAS + MArch） | 5年 |
| **执照** | OAA（安大略建筑师协会）注册建筑师（高中毕业后约9–11年） | P.Eng.（专业工程师，高中毕业后约8–9年） |
| **入职薪资** | $50,000–$70,000 | $62,000–$75,000 |
| **Co-op时薪** | $21–27/小时 | $21–27/小时 |
| **雇主需求** | 创意设计领袖；项目建筑师 | 建筑系统工程师；结构设计师；BIM（建筑信息模型）专家 |
| **权衡取舍** | 更大的创意自由度；执照申请路径更长、更艰难 | 更扎实的技术深度；执照申请更快；起始薪资略高 |

## 3. 入学要求

| 因素 | **建筑学（BAS）** | **建筑工程学（BASc）** |
|--------|------------------------|--------------------------------------|
| **必修课程** | ENG4U（75%+）、高等函数（70%+）、微积分（70%+）、物理（70%+） | ENG4U、高等函数、微积分、物理、化学 |
| **竞争性均分** | **80分段低至中段**（约82–86%） | **80分段高段**（约87–92%） |
| **需提交作品集？** | **是**——视觉艺术/设计作品 | **否** |
| **是否面试？** | **是**——四月在剑桥校区举行 | 否 |
| **需提交AIF（入学信息表）？** | 否 | **是** |
| **录取难度** | 高度竞争（艺术+成绩） | 竞争激烈（成绩+AIF） |
| **录取对象** | 具备设计才能且成绩优秀的学生 | 理工科成绩突出、对工程有兴趣的学生 |

**核心要点**：建筑学考查*创意能力*，建筑工程学考查*理工科学业成绩*。两个专业均要求学生在各自领域表现出色，但筛选机制截然不同。

## 5. 课程：您将学到什么

### 共同基础

两个专业均在以下方面投入大量学时：
- **建筑结构体系**（荷载、重力、侧向力、基础）
- **设计工作室**（两个专业每学期均有）
- **建筑技术**（外围护结构、材料、建造工艺）
- **可持续性与环境设计**
- **数字工具**（两者均教授Revit、AutoCAD及计算方法）

### 分化方向

| 领域 | **建筑学** | **建筑工程学** |
|------|-----------------|------------------------------|
| **设计工作室比重** | *主导地位*——设计统领一切决策 | *融合地位*——设计贯穿始终，但工程逻辑主导 |
| **结构分析深度** | 中等（足以与工程师协作） | 深入（与土木工程专业同等深度） |
| **建筑机电系统（MEP）** | 概览层级 | 深度专业化方向可选（建筑系统专项） |
| **建筑历史与理论** | 广泛——贯穿5年完整学习 | 仅一门概论课 |
| **数学深度** | 中等（微积分，部分线性代数） | 深入（与完整工程学相同：常微分方程、线性代数、数值方法） |
| **编程/计算方法** | Rhino/Grasshopper（参数化设计） | Python/MATLAB（工程分析） |
| **物理/力学** | 应用层面（静力学、结构行为） | 完整工程力学课程序列 |
| **人文社科内容** | 丰富（历史、理论、城市主义） | 极少 |
| **第三年学习地点** | 全年驻留剑桥校区；深度设计浸入 | 在剑桥校区学习一年（与建筑学MArch学生同校） |
| **第四年** | 意大利罗马（秋季学期） | 滑铁卢主校区 |
| **专业化选项** | 无正式分流——通科建筑师路径 | 建筑结构专项 或 建筑系统专项 |
| **是否需要研究生学位？** | **是**——执照申请需要MArch | **否**——BASc即为P.Eng.申请的终极学位 |

### 核心区别，简而言之

**建筑学**：您被培养成*建筑的作者*——将场地、功能、文化、结构与美学整合为连贯的设计理念。您主导项目，是首席创意决策者。

**建筑工程学**：您被培养成*建筑的技术实现者*——确保建筑愿景在结构上安全、能耗上高效、法规上合规、施工上可行。您与建筑师并肩工作。

在实践中，这两种角色存在大量交集，任一领域的顶尖专业人士都对双方有深入理解。但两者的培养侧重确实存在本质差异。

## 6. 职业执照申请路径

### 建筑学 → OAA（安大略建筑师协会）

```
BAS（5年）
    ↓
MArch（滑铁卢大学1–2年；BAS成绩75%以上可直接录取）
    ↓
CACB（加拿大建筑认证委员会）认证（学历资质获认可）
    ↓
OAA实习建筑师注册（IAP实习生入职计划）
    ↓
累计2,800小时CERB工作经历（约1.5–2年，须在注册建筑师监督下全职工作）
    ↓
安大略执业能力自我评估（2025年新增）
    ↓
ExAC（每年一次；2024年通过率72.2%）
    ↓
OAA入会课程
    ↓
OAA执照：注册建筑师 ✓

自高中毕业起总计：约9–11年
```

### 建筑工程学 → P.Eng.（安大略专业工程师执照，由PEO颁发）

```
建筑工程学BASc（5年，CEAB认证）
    ↓
NPPE（全国专业执业考试——涵盖职业道德、法律与执业实践）
    ↓
48个月监督下工程工作经验
（第二年后的Co-op学期最多可折算12个月）
    ↓
PEO基于能力的评估（CBA）
    ↓
良好品格审查
    ↓
P.Eng.执照：专业工程师 ✓

自高中毕业起总计：约8–9年
（Co-op折算后，毕业后实际所需工作年限约为3年）
```

### 路径对比

| 因素 | **OAA（建筑学）** | **P.Eng.（建筑工程学）** |
|--------|------------------------|---------------------------------------|
| 本科后是否需要额外学位 | 是——须获得MArch | 否 |
| 所需工作经验年限 | 约2–3年（2,800小时CERB） | 4年（48个月，其中Co-op可折算12个月） |
| 资格考试 | ExAC（每年一次；通过率72.2%） | NPPE（相对简单；通过率约85%） |
| 自高中毕业起总年限 | **约9–11年** | **约8–9年** |
| 持证后可执业范围 | 设计并签署*建筑*图纸 | 签署并盖章*工程*图纸及报告 |
| 能否从事另一专业的工作？ | 否（须另获工程学位） | 否（须另获建筑学学位） |
| 是否需要额外课程 | 是，须完成OAA入会课程 | 无（NPPE之外不需要） |
| 考试频率 | 每年一次 | 每年多次 |
| 自雇比例 | 24% | 约10% |

**关键洞察**：两条路径均较漫长。P.Eng.快1–2年，考试也更简单。OAA需要更多教育投入，但授予"建筑师"称号——这是一个在设计领域具有更高社会认可度的受保护头衔。

| BIM市场增长 | 年复合增长率11.3–14.6%；两个专业均教授BIM |
| 安大略省建筑劳动力缺口 | 仅安大略省一地2034年前将短缺52,000名工人 |

| 绿色岗位（加拿大） | ✅ 预计每年75,000个 | ✅✅ 每年75,000个；建筑工程学在技术类绿色岗位上更占优势 |
| BIM需求 | 高 | **极高**；建筑工程学毕业生常成为BIM负责人 |

### AI影响：一个关键区别

两个专业均面临AI冲击，但方式不同：

**建筑学 + AI：**
- AI可生成设计方案变体、渲染图像、运行能耗模拟
- 初级建筑师（从事制图/生产性工作）面临最大替代风险
- 善用AI进行创意构思、客户沟通和复杂设计判断的建筑师处境更稳
- 风险：若AI取代入门级生产性工作，初级建筑师将如何积累技能？

**建筑工程学 + AI：**
- AI可自动化规范合规检查、碰撞检测、能耗模拟运算
- 但法规要求注册工程师须在设计上签字盖章——AI无法取代P.Eng.印章
- 技术判断与法规解读仍以人为主导
- 替代风险较低，因为注册工程师受到法规保护，AI无法绕过

**AI影响结论**：在AI替代浪潮下，建筑工程学具有轻微的结构性优势——P.Eng.法规要求为注册工程师的需求提供了兜底保障，而建筑学在这方面没有同等程度的保护。

## 9. 学生生活与校园文化

### 建筑学（剑桥校区 + 罗马）

- 高强度、封闭式工作室文化；您将与同届约70名同学朝夕相处
- **剑桥校区**更像一所设计学校，而非综合性大学——工业风建筑、宽阔河景、紧密的社群氛围
- 工作室文化意味着频繁的深夜赶图、频繁的评图（Crit），以及对设计质量的高度同伴压力
- 第四学年秋季赴罗马的经历具有深刻的启迪意义，但费用高昂、后勤复杂
- 频繁的嘉宾讲座与访问评图人；与全球建筑话语保持密切联系
- 与滑铁卢主校区相比，社会多样性较低（社群规模小，地理位置相对偏僻）
- 学生形容这段经历既是"人生中最精彩的时光"，也是"令人精疲力竭且竞争激烈的过程"

### 建筑工程学（滑铁卢主校区 + 第三年剑桥校区）

- 第一、二、四、五年在滑铁卢主校区接受工程学院文化熏陶
- 第三年赴剑桥校区，体验建筑学工作室文化——是短暂的少数派经历，之后返回主校区
- 与其他工程专业联系更紧密；主校区社交生活更丰富
- 每学期均有设计工作室，提供比典型工程项目更多的创意挑战
- 与多数滑铁卢工程专业相比人数较少（每届约85人）
- 在工程学标准下，文化氛围协作友好、设计导向，同时兼具技术严谨性

### 关键差异

建筑学学生的身份认同首先是**设计师与创意专业人士**。建筑工程学学生的身份认同首先是**理解设计的工程师**。两者都上设计工作室，都关注建筑。但这一根本框架的差异深刻影响着您对职业的自我定位。

## 10. 费用与财务对比

### 项目总费用估算（安大略省本地学生）

| 费用项目 | **建筑学（BAS+MArch）** | **建筑工程学（BASc）** |
|----------------|------------------------------|--------------------------------------|
| BAS/BASc学费（每年） | 约$13,000 | 约$13,000–$15,000 |
| 缴纳学费年数 | 5年（BAS）+ 1–2年（MArch）= **共6–7年** | **5年** |
| MArch学费（总计） | 约$39,941 | 不适用 |
| 工作室耗材/材料/年 | 约$8,100（第一年）；后续年份较少 | 约$2,000–$4,000/年（软件、材料） |
| 罗马学期额外费用 | 约$5,000–$10,000（生活费、机票） | 不适用 |
| **学位总费用估算** | **约$130,000–$160,000+** | **约$80,000–$100,000** |
| Co-op收入（可抵扣） | 约$75,000–$85,000 | 约$80,000–$95,000 |
| **净估算费用** | **约$55,000–$85,000** | **约$0–$20,000** |

> 以上为粗略估算。建筑工程学项目在财务上明显更为精简——Co-op收入往往可覆盖相当大比例的学费。建筑学额外的MArch学费、工作室耗材费及罗马旅行费用，使总投入明显更高。

### 投资回报率（ROI）分析

| 指标 | **建筑学** | **建筑工程学** |
|--------|-----------------|------------------------------|
| 首次获得完整薪资所需年数 | 6–7年（MArch毕业后） | 5年（BASc即为终极学位） |
| 入职薪资 | $50–70K | $62–75K |
| 距获得持证后中位薪资所需年数 | 自高中起约9–11年 | 自高中起约8–9年 |
| 持证后中位薪资 | 约$85,300/年 | 约$99,847/年 |
| 投资回报周期 | 较长；前期投入更高 | 更快；前期成本更低 |

**财务结论**：建筑工程学在财务效益上明显更优——成本更低、达到高薪的路径更快、中位薪资更高。建筑学的投资回报在很大程度上取决于能否晋升至高级/合伙人级别，或转型进入开发等相关领域。

## 11. 决策框架：哪个专业适合您？
